# 网络安全
## 计算机网络通信面临4种威胁
* 被动攻击
    * 截获（interception）: 攻击者从网络上窃听他人的通信内容
* 主动攻击
    * 中断（interruption）: 攻击者有意中断他人在网络上的通信
    * 篡改（modification）: 攻击者故意篡改网络上传送的报文
    * 伪造（fabrication）: 攻击者伪造信息在网络上传送

## 两类密码体制
* 对称秘钥密码体制
    * 加密与解密密钥相同
    * 例如：高级加密标准AES，数据加密标准DES、国际数据加密算法IDEA
* 公钥密码体制
    * 加密与解密密钥不同
    * 例如：RSA体制
